http methods

1. get: any inforamtion taken is served  to the server to the   frontend. 
2 post: any inforamtion taken from frontend by the server
3. put: any inforamtion updated  by the server to the frontend  
4. delete: any inforamtion deleted by the server  to the frontend    
5. patch: specific inforamtion updated by the server to the frontend
